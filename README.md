# Projet de création de Blog

Technologies utilisées pour la création du blog :
* [Symfony](https://symfony.com/)
* [Bulma](https://bulma.io)

Le blog est hébergé sur Heroku à l'adresse [BlogProject](https://blogproject43.herokuapp.com)

## Compte user

Peut ajouter et supprimer ses commentaires sur les articles.

## Compte admin

Vous pouvez tester les fonctionnalités administrateurs tels que :
Ajouter, modifier & supprimer un article, supprimer les commentaires de tout utilisateur, ajouter et supprimer des tags.

Compte admin :
* username : admin
* mot de passe : password
